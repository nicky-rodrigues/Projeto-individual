# Hiraeth

Hiraeth é uma rede social literária desenvolvida como projeto individual, com foco em leitores que desejam registrar leituras, explorar gêneros literários e descobrir novas recomendações em um ambiente inspirado no universo dos livros, das bibliotecas e da fantasia.

A proposta do projeto é criar uma plataforma que vá além de uma página comum sobre livros, oferecendo um espaço acolhedor e visualmente imersivo para pessoas que encontram nos livros um lugar de pertencimento. A tela inicial foi pensada como uma apresentação desse universo, enquanto a proposta da plataforma, após o login, é funcionar como uma rede social voltada à experiência literária do usuário.

## Objetivo do projeto

O objetivo deste projeto é aplicar os conhecimentos desenvolvidos ao longo do semestre em um site autoral, unindo HTML, CSS, JavaScript, banco de dados e integração com Node.js. Além disso, o projeto busca representar um tema pessoal e construir uma interface que traduza visualmente essa identidade.

## Funcionalidades pensadas para a plataforma

- Tela inicial com identidade visual inspirada em bibliotecas e fantasia
- Cadastro e login de usuários
- Exploração de gêneros literários
- Registro de leituras
- Dashboard literária com informações sobre a jornada de leitura do usuário

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Node.js
- Banco de Dados

## Identidade visual

A identidade visual do projeto foi inspirada na estética dark academia, em bibliotecas antigas e no universo da fantasia. A paleta de cores utiliza tons de marrom escuro, bege pergaminho, dourado envelhecido e preto suave.
